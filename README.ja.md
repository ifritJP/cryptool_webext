# Cryptool

[English Version](readme.md)

暗号化されたメモを安全に閲覧するための、ローカル完結型ビューアーです。AES-256 (GCM) および PGP 対称鍵暗号に対応しています。

## 主な機能

- **ローカル復号**: すべての復号処理はブラウザ内で行われます。データがサーバーに送信されることはありません。
- **複数のフォーマットに対応**:
  - 独自 AES-256 GCM (PBKDF2 鍵誘導を使用)。
  - 標準 PGP 対称鍵暗号 (AES-256, SHA-512 S2K)。
- **セキュリティ機能**:
  - **非表示時の自動クリア**: タブがバックグラウンドに移動したり閉じられたりした際、機密データを自動的に消去します（設定で変更可能）。
  - **メモリ安全性**: 使用後、メモリ上の機密データを積極的にクリアします。
- **プレミアムな UI**: グラスモーフィズムを採用したモダンなダークテーマと、スムーズなインタラクション。
- **カスタマイズ**: 読みやすさに合わせてフォントサイズを調整可能。
- **安全なキー入力**: パスワードの表示/非表示を切り替え、安全性を保ちつつ利便性を確保。

## インストール方法

1. このリポジトリをクローンします。
2. ブラウザの拡張機能管理ページ（`chrome://extensions` または `about:debugging`）を開きます。
3. 「デベロッパー モード」を有効にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`src` ディレクトリを選択します。

## 使い方

1. ブラウザのツールバーにある Cryptool アイコンをクリックします。
2. **"Open Viewer"** を選択します。
3. 暗号化ファイルを選択（またはドラッグ＆ドロップ）します。
4. 復号キーを入力します。
5. 必要に応じてフォントサイズを調整します。

## 技術仕様

- **Manifest V3**: 最新のブラウザ拡張機能標準に準拠。
- **Web Crypto API**: モダンな AES-GCM 復号に使用。
- **OpenPGP.js**: 堅牢な PGP サポートのために使用。
