# Cryptool 仕様書

## 概要
AES-256 および PGP 共通鍵暗号化されたメモファイルを安全に閲覧するための Web 拡張機能。

## 機能要件
- **対応フォーマット**:
    - AES-256 GCM (PBKDF2 鍵派生)
    - OpenPGP Symmetric Encryption (AES-128/192/256, SHA-1/256/384/512 S2K 対応)
- **復号プロセス**:
    - ブラウザ内（ローカル）ですべての復号処理を行う。
    - 外部サーバーへのデータアップロードは一切行わない。
- **UI/UX**:
    - ダークモードに対応したモダンなグラスモーフィズムデザイン。
    - 拡張機能アイコンクリック時のメニューから、独立したタブ（Viewer）を起動。
    - 復号鍵（パスワード）の表示/非表示切り替え。
    - メモ閲覧時のフォントサイズ調整（スライダーによるリアルタイム変更）。
- **制約事項**:
    - 閲覧専用（Viewer）。編集・保存機能は持たない。
    - 範囲選択およびコピーはサポートする。
    - 拡張機能内にパスワードや平文などの情報を一切保存しない。

## 技術スタック
- **プラットフォーム**: Web Extension (Manifest V3)
- **ブラウザ互換性**: Firefox (Gecko), Chrome (Chromium)
- **暗号化ライブラリ**:
    - Web Crypto API (AES-GCM)
    - OpenPGP.js (PGP Support)

## ディレクトリ構造
- `src/`: 拡張機能ソースコード
- `src/libs/`: 外部ライブラリ
- `docs/`: 仕様書および作業記録
- `scratch/`: テスト用スクリプト等
